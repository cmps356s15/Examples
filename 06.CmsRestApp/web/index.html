REST Service - Contact 1

You can test other examples using Postman Chrome App - http://www.getpostman.com/

Get Contacts:

Get request for Url: http://localhost:8080/cmsapp/api/contacts
You can get either json if accept="application/json" or XML if accept="application/xml"

Get a contact:

Url: http://localhost:8080/cmsapp/api/contacts/1

Add Contact:

Url: http://localhost:8080/cmsapp/api/contacts using Post request
You can post json by setting the Content-Type="application/json"
{"firstName":"Ali","lastName":"Saleh","address":{"street":"15 Fun St","city":"Doha","country":"Qatar"}}

Update contact:

Url: http://localhost:8080/cmsapp/api/contacts using Put request
You can post json by setting the Content-Type="application/json"
{"contactId":1,"firstName":"Samir","lastName":"Ali","address":{"addressId":1,"street":"15 History St","city":"Fes","country":"Morocco"}

Delete contact:

Url: http://localhost:8080/cmsapp/api/contacts/1 using Delete request
